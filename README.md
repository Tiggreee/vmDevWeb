# vmDev
Landing portfolio con scroll horizontal y estetica futurista. Presenta proyectos, servicios, skills y un flujo de contacto con modal y feedback visual.

## Caracteristicas
- Navegacion horizontal por secciones con scroll snap.
- Filtros de proyectos por categoria.
- Modal de contacto y toast de confirmacion.
- Micro-interacciones en el hero y botones.

## Tecnologias
- HTML
- CSS
- JavaScript (vanilla)

## Ejecutar
Opcion simple:
1. Abre index.html en tu navegador.

Opcion con servidor local (recomendado):
1. Instala un servidor estatico (por ejemplo, Live Server).
2. Inicia el servidor desde la carpeta del proyecto.

## Estructura
- index.html: markup principal y secciones.
- styles.css: estilos, layout horizontal y animaciones.
- main.js: interacciones (modal, filtros, scroll).
- assets/: recursos visuales.
